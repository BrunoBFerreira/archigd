procedure

(wall p0 p1 [bottom-level top-level family]) → wall-shape

  p0 : Loc
  p1 : Loc
  bottom-level : Level = (current-level)
  top-level : Level = (upper-level bottom-level)
  family : Any = (default-wall-family)
Creates a wall given two points p0 and p1. Returns a wall that can be used by other operations, such as window, door, among others. All other arguments are optional, and have default values.

Parameter bottom-level determines the bottom level of the created wall.

Parameter top-level determines the top level of the created wall, it will link the wall to that level and control its height.

Parameter family contains information regarding the wall’s type. At the moment only thickness is supported.

Example:
> (wall (x 0)(x 10))

procedure

(slab vertices [level family]) → slab-shape

  vertices : Locs
  level : Level = (current-level)
  family : Any = (default-wall-family)
Creates a slab given a list of points, guide. The guide is not closed, i.e., the last point does NOT have to be equal to the first.

Parameter level determines the level of the created slab. This will also determine the slab’s height, i.e., it will be the same height as the level in which the slab is placed.

Parameter family contains information regarding the slab’s type. At the moment only thickness is supported.

procedure

(window wall relative-pos) → window-shape

  wall : wall-shape
  relative-pos : number?
Creates a window in a given wall at a position relative to that wall, relative-pos. The parameter relative-pos is the middle-point of the window. Furthermore it is a distance relative to the initial point of the wall, relative-pos = 0. For a wall with a length of 5, and to position a window in the middle one would give relative-pos as 2.5.

procedure

(level height) → level

  height : number?
Creates a level at a given height. If the level already exists, it returns that level and there is no creation.

procedure

(upper-level [lvl height]) → level

It creates a level at given height based on a given level, lvl. If the level already exists, it returns the level and no level is created.
